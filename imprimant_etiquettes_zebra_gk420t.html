Imprimant d'étiquettes zebra gk420t

Les imprimantes G-Series de Zebra sont compactes et offrent les meilleures vitesses et performances de leur catégorie pour une largeur d’impression allant jusqu’à 104 mm. Dotées de nombreuses fonctionnalités, Tous les modèles proposent en natif les olangages EPL™ et ZPL® et de nombreuses options de connexions pour des systèmes personnels, locaux et en réseau. Imprimante codes barres de bureau. La zebra GK420 T (Transfert Thermique) est idéale pour de petit volume d'impression. L'imprimante GK420T de ZEBRA peut se piloter via le logiciel Zebra Designer. Ce logiciel, gratuit, permet la création simple d'une étiquette code barres. Fiche technique : • Volume d'impression /jour : 500 • Largeur d'impression (mm) : 104 • Résolution (DPI) : 203 • Résolution (DOT) : 8 • Type d'impression : Transfert Thermique • Interface de communication : USB, Ethernet • Diamètre mandrin étiquettes (mm) : 12,7 ou 25,4 ou 37,1 • Vitesse d'impression (mm/s) : 127
